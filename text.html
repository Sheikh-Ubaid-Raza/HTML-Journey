Text Formatting Elements:

Headings

HTML has six levels of headings.
h1
h2
h3
h4
h5
h5
h6

h1 is used for main heading.
h2,h3,h4,h5,h6 is used for subheadings

Paragraphs

To create a paragraph,surround the word that make up the paragraph with an opening p and closing /p tag.

Bold & Italic

By enclosing words in the b tag we can make characters Bold.

By enclosing words in the i tag we can make characters Italic.

Superscript & Subscript

The sup element is used to contain characters that should be superscript.For example: 22

The sub element is used to contain characters that should be subscript.For example: H2O

Line breaks & Horizontal rules

If you want to add a line break inside the middle of a paragraph you can use the line break br tag

If you want to add a horizontal line in the middle of a paragraph you can use the horizontal rule hr tag


List

Types of Lists

There are three types of lists in Html page

  1. Ordered List
  2. Un-ordered List
  3. Definition List

Ordered Lists

Ordered lists are list where each item in the list is numbered.
The Ordered lists is created with the ol element.

List item

Each item in the ordered list is placed between an opening li tag and a closing/li tag.
(li stands for List item)

Un-ordered Lists

Un-ordered List are list that begin with a bullet point(rather than character that indicate order).
The Un-ordered lists is created with the ul element

List item

Each item in the Un-ordered list is placed between an opening li tag and a closing/li tag.
(li stands for List item)

Definition Lists

Definition Lists are made up of a set of terms along with the definitions for each of those elements.
The Definition List is created with the dl element and usually consists of a series of terms and their definitions.

Inside the dl element you will usually see pairs of dt and dd elements

dt

This is used to contain the term or heading being defined.
(dt stands for definition term)

dd

This is used to contain the definition.
(dd stands for definition data)
